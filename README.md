# indigo

A place for technology previews related to scribus

##

~~~sh
$ cd indigo/
$ mkdir build
$ cd build/
$ cmake ..
$ make
$ ./indigo/indigo
~~~

Some ideas:

- a plugin API
- compiled plugins
- a python scripting engine
- a d-bus interface
- a clean separation between gui and non gui
- an html/css-like file format
